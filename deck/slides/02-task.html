1 · Intro — Task

Compose a pretrained image model with a pretrained video model, and learn only the temporal residual.

Given a source video X = (x₁…x_T) and an edit instruction p, produce an edited video Y that is (i) faithful to p on every frame and (ii) temporally consistent — under a causal, streaming, constant-latency constraint. We do not learn this map end-to-end. We factor it.

The factorization

A frozen per-frame image editor E gives a draft Ỹ = (E(x₁,p), …, E(x_T,p)). Each frame is structurally exact — every car, sign and pole stays in place — but the frames are drawn independently, so the draft flickers.

The learned model F (the fixer) maps the flickery draft back onto the temporally consistent manifold while keeping the draft's appearance: Y = F(Ỹ, X), causal in t.

The reason this is worth doing is an entropy argument: H(Y*|X,p) ≫ H(Y*|Ỹ,X,p). Once the image model has fixed the semantics, what remains is a low-entropy, frame-to-frame residual — and that is a far easier learning problem than video editing.

What is being learned

A denoiser whose only noise model is temporal flicker. Not an editor. Not a smoother. Content is out of scope by construction; the residual is the whole task.

Deployment constraints

  • Causal: frame t may not see t+1.
  • Streaming: chunk-by-chunk, constant memory at any length.
  • Single step: latency budget is a fraction of one chunk.
  • Domain: driving video (pass-through AR in vehicles).
Formally: learn F such that F(Ỹ, X) ≈ argmin_Y d_time(Y)  s.t.  d_appearance(Y, Ỹ) ≤ ε, causally. 02
