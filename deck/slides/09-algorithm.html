2 · Algorithm

Edit-anchored rectified flow: start at the draft, not at noise.

The path

Standard rectified flow travels from Gaussian noise to data. We re-anchor the σ = 1 endpoint to the draft:

a  = z̃ + σ_a · ε
x_σ = (1−σ)·z* + σ·a
v  = a − z*

z̃ = draft latent, z* = clean target, σ_a = 0.1. The model regresses the velocity v; the clean estimate ẑ falls out in closed form.

The losses — and what each one buys

  • Flow matching — the fix itself.
  • Residual anchor λ|ẑ − z̃|₁ — "you may change time, you may not change appearance". This is the first gate against erasing the edit.
  • Latent high-pass — retains edges. Measured to be a temporal↔quality dial, not a sharpness fix; we set it deliberately, not greedily.

Why a single step is enough interactive — drag σ

The travel distance is short by construction, so the flow does not need to be integrated finely. Measured: K=1 removes 56.9% of flicker, K=4 removes 56.1%. Identical — so we take the cheap one.
